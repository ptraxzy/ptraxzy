## About

I am Muhammad Putra, a full-stack developer from Malang, Indonesia. I build web applications, AI-focused product interfaces, automation bots, and practical tools using the technologies visible across my public repositories.

My current work is centered around TypeScript web products, PHP/MySQL applications, Python automation, JavaScript bot workflows, and the Ultramaxo ecosystem.

- Building AI web products and chat-oriented interfaces.
- Developing full-stack applications with TypeScript, PHP, Python, JavaScript, HTML, and Blade.
- Creating automation workflows for Telegram, WhatsApp/Baileys, downloader tools, and real-time monitoring.
- Keeping projects public, iterative, and easy to review from GitHub activity.
