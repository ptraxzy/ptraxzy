<h2 align="center">About</h2>

I am Muhammad Putra, a self-taught full-stack developer from Malang, Indonesia, focused on building AI-powered products, clean web applications, automation systems, and practical developer tools. My work combines software engineering fundamentals, modern TypeScript stacks, backend architecture, and product-minded execution to turn fast ideas into usable systems.
